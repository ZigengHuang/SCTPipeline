【临床关键错误纠正指南】

一、常见医学近音/形近混淆词对照表（ASR高频错误）
错误词 → 正确词（使用语境）
约 → 药（"吃约"→"吃药"，"泻约"→"泻药"，"换约"→"换药"，"麻醉约"→"麻醉药"，"消炎约"→"消炎药"）
药 → 约（"预药"→"预约"，仅在预约/约定语境下）
检验 → 检查（当指影像检查、内镜检查、体格检查时："做检验"→"做检查"，"CT检验"→"CT检查"，"B超检验"→"B超检查"）
检查 → 检验（仅当指实验室化验时：如"血液检查"→"血液检验"）
迁 → 签（"迁字"→"签字"，"迁麻醉"→"签麻醉"）
作 → 做（"作手术"→"做手术"，"作检查"→"做检查"）
发痛 → 发炎（"伤口发痛"→"伤口发炎"，在炎症语境下）
按痛 → 按压（"按痛30分钟"→"按压30分钟"，在止血/穿刺后语境下）
头痛 → 头晕（当描述麻醉/术后反应时，注意区分具体症状）
车线 → 拆线（"车线"→"拆线"）
唐正军 → 糖浆（药名语境下）
阿莫西宁 → 阿莫西林
头饱 → 头孢
青霉素 ≠ 庆大霉素（注意区分，不可互换）
胃经 → 胃镜
肠经 → 肠镜
消炎要 → 消炎药
止痛要 → 止痛药

二、否定词关键检查规则
- 医嘱中的否定词（不/没/无/别/非/勿）必须准确保留，不可随意增删
- 高危否定错误示例：
  "没有过敏史" ≠ "有过敏史"
  "不需要空腹" ≠ "需要空腹"
  "术后不让下地" ≠ "术后让下地"
  "不用停药" ≠ "用停药"
  "别吃辛辣" ≠ "吃辛辣"

三、数值/剂量合理性参考
- 口服药物单次剂量通常为mg级别（1-500mg），若出现数千mg需警惕
- 输液量通常500-2000ml，若出现20000ml需警惕
- 泻药一般1-3包，若出现"二十包"需警惕
- 体温报警阈值通常37.3-38.5℃，若出现39.5℃作为报警阈值需警惕
- 服药频次通常一天1-3次，超出此范围需结合上下文验证

四、时间表述合理性参考
- 术后平卧时间通常2-8小时，若出现"分钟"单位需警惕
- 换药间隔通常3-7天，若出现"一天换三次"需结合语境判断
- 拆线时间通常7-14天，若出现明显偏离需警惕
- 复查间隔通常1周-3个月，若"一个月"被替换为"一天"需警惕
- 禁食时间通常4-12小时，注意"小时"与"分钟"的区别
